# Celo MiniPay App

## 📌 Project Overview
An innovative mobile finance application built for the Celo MiniPay ecosystem. This project leverages stablecoins and mobile-first infrastructure to provide seamless, low-cost financial utility.

## 🚀 Features
* **MiniPay Integration:** Optimized for fast, secure microtransactions.
* **Stablecoin Support:** Designed to handle digital asset transactions smoothly.
* **Mobile-First UX:** Tailored specifically for mobile web browser environments and Termux-based deployment.

## 🛠️ Tech Stack
* **Blockchain:** Celo Network
* **Platform:** MiniPay
* **Configuration:** Git, Node.js environment

## 📁 Repository Structure
* `SKILL.md`: Core project specifications and skill mapping.
* `skills-lock.json`: Dependency locks for the project build environment.
